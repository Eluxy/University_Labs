# Java Lab

Сборник учебных работ по Java: лабораторные задания, практика по ООП, коллекциям, обработке исключений и небольшие отдельные мини-проекты.

## Содержание репозитория

- `src/` — базовые лабораторные (`Lab1`, `Lab2`, `lab3`, `Lab4`, `LAb5`) по основам Java.
- `Lab7/` — ООП: классы, наследование, интерфейсы (животные, транспорт, профессии).
- `Lab8/` — коллекции Java (`ArrayList`, `LinkedList`, операции с элементами и списками).
- `lab9/` — обработка исключений и простые задания по этой теме.
- `KR/` — контрольная/практика по работе с сущностью автомобиля (`Car`, `CarManagement`).
- `Kurs/` — курсовые/дополнительные задачи, включая парсер арифметических выражений.

## Требования

- JDK 17+ (подойдет и JDK 11+ для большинства файлов).
- IntelliJ IDEA (рекомендуется, проект изначально собирался в IDEA).

## Как запустить

### В IntelliJ IDEA

1. Откройте папку проекта.
2. Выберите нужный класс с `public static void main` (например, `src/Lab1.java`).
3. Запустите через кнопку **Run**.

### Через командную строку (Windows PowerShell)

Пример для `Lab1`:

```powershell
javac .\src\Lab1.java
java -cp .\src Lab1
```

Пример для `CarManagement` из папки `KR/src`:

```powershell
javac .\KR\src\Car.java .\KR\src\CarManagement.java
java -cp .\KR\src CarManagement
```

## Примеры точек входа

- `src/Lab1.java` — базовый ввод/вывод и арифметика.
- `src/Lab4.java` — методы и перегрузка.
- `src/LAb5.java` — массивы и поиск/сортировка.
- `Lab7/src/Main1.java` — демонстрация ООП-моделей.
- `Lab8/src/Main3.java`, `Lab8/src/main2.java` — практика с коллекциями.
- `lab9/src/Main4.java` — обработка ошибок ввода.
- `KR/src/CarManagement.java` — операции со списком машин.
- `Kurs/src/Main.java` — вычисление арифметических выражений.

## Примечание

Проект учебный, поэтому структура и стиль кода в разных папках могут отличаться (разные этапы обучения и отдельные задания).
